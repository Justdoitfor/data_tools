# 招投标信息自动化采集与分析系统

## 简介
这是一个支持从主流招投标平台批量抓取项目信息，并自动进行数据清洗、多维排序、关联分析和商机推荐的自动化工具与展示平台。

## 功能特性
- **数据采集**：支持关键词、地区、时间等多维度的自动化数据抓取（内置模拟数据与反爬架构设计）。
- **数据清洗**：剔除重复项目，结构化金额、时间等核心字段。
- **关联分析**：高频中标单位排行（竞品）、高频招标单位排行（大客户）。
- **趋势分析**：按月/季度统计招投标活跃度，自动划分金额区间。
- **Web 可视化**：提供基于 `Streamlit` 的开箱即用的实时交互数据大屏。

## 安装指南
```bash
# 1. 进入项目目录
cd tender_bot

# 2. 安装依赖
pip install -r requirements.txt
```

## 使用方法

### 1. 命令行 CLI 工具
提供轻量级抓取，结果自动保存为 CSV：
```bash
# 运行抓取命令
python cli.py --scrape -k "大数据" -r "北京" -l 20 -o result.csv
```

### 2. 交互式 Web 看板
生成初始数据并启动可视化大屏，支持在线搜索与图表展示：
```bash
# 首先生成初始测试数据 (仅用于无数据库测试场景)
python generate_sample.py

# 启动 Web 服务
streamlit run app.py
```
启动后，访问浏览器显示的本地端口（通常为 `http://localhost:8501`）即可使用。
